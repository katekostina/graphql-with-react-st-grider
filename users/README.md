###### 1. Start a database:
`npm run json:server`

it will be on the 3000 port:
> http://localhost:3000

###### 2. Start a server 
with nodemon:

`npm run dev`

without nodemon:

`npm run start`

###### 3. Play with queries
go there and play with GraphiQL

<http://localhost:4000/graphql>
